# 📝 Write — Minimalist Writing Pad

A distraction-free, elegant writing space built with pure HTML, CSS, and JavaScript.

Designed for writers who just want to **think, type, and flow**—nothing more, nothing less.

---

## ✨ Features

* 🌿 **Minimal Interface**
  No toolbars, no buttons—just a clean writing canvas.

* 🌓 **Auto Theme (Light/Dark Mode)**
  Adapts to your system theme for a comfortable writing experience.

* 💾 **Auto Save (Smart Saving)**
  Saves your content automatically after you stop typing for a few seconds.

* 🧹 **Auto Clear When Empty**
  If you delete everything, the app removes saved content from local storage.

* 📊 **Live Word Count**
  Subtle word counter that appears while typing.

* 🎯 **Focus First Design**
  Carefully chosen typography and spacing for long writing sessions.

---

## 🚀 How It Works

* Start typing—your words are automatically saved.
* Stop typing for ~2 seconds → content is saved.
* Delete everything → storage is cleared.
* Reload the page → your last content is restored (if any).

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (with system theme support)
* Vanilla JavaScript (no frameworks)

---

## 📂 Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/TwilightEntity/write.git
   ```

2. Open the project:

   ```bash
   cd write
   ```

3. Run:

   * Simply open `index.html` in your browser.
